About this project

Ever since my 3D-printed skull project from two years prior, I knew I wanted to complete the set by printing my brain life-sized. I first got an MRI from a study I did at TODO, which gave me a nii file, which is the file format typically used by radiologists' programs. From there, I created a virtual machine in Linux and installed TODO, which allowed me to convert the nii file to an STL file.

After pulling the STL, I needed to smooth out the noise, so I sculpted it using blender. Finally, I was able to print the brain in a Fuse 1+ at Formlabs, then sandblast and polish it. Here's what the final print looked like!
